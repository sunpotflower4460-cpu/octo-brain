## 1. 取得する情報

OctoBrain(以下「本アプリ」)は、アカウント登録なしで利用できます。次を取り扱います。

- **入力テキスト**: あなたが分析のために入力した文章。回答生成のために外部の大規模言語モデル(LLM)API へ送信されます。
- **端末内データ**: 会話履歴・下書き・設定は、**あなたの端末内(ローカルストレージ)にのみ保存**されます。当社サーバーには保存しません。
- **利用量の識別子**: 連打防止・月間利用回数の管理のため、端末側で生成したランダムな識別子(clientId)を用います。氏名・メールアドレス等の個人情報とは結びつきません。
- **コスト・利用量ログ**: 品質と原価の監視のため、モデル呼び出しの**トークン数・概算コスト・処理時間**をサーバーに記録します(**入力・出力の本文は含みません**)。保持期間は最大 [90] 日です。

本アプリは、広告SDK・解析SDKによる**トラッキングを行いません**([課金/広告を追加する場合はこの記述を更新])。

## 2. 外部送信先(LLM プロバイダー)

回答生成のため、入力テキストは処理目的で外部 LLM API に送信されます。利用中のプロバイダー: [要記入(例: DeepSeek 等、`workers/src/config/models.ts` で確定したもの)]。各プロバイダーの取り扱いは各社のポリシーに従います。当社は入力本文を分析処理以外の目的に利用しません。

## 3. 利用目的

- 入力に対する分析回答の生成
- 不正・過負荷の防止(連打防止・利用回数上限)
- 品質・原価・障害の監視と改善

## 4. 第三者提供

法令に基づく場合を除き、本文データを第三者へ販売・提供しません。LLM プロバイダーへの送信は「回答生成のための処理委託」に該当します。

## 5. データの保存と削除

- 会話・設定は端末内にあり、アプリの「ローカル会話データを削除」または端末側の操作で削除できます。
- サーバーのコスト/利用量ログは本文を含まず、[90] 日で自動失効します。
